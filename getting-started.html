Getting started

1. Create

Study the technical details and advices, then create your assets and animations as usual in Flash Professional.

Download Flash Professional

2. Publish

Publish to Dart code using the Toolkit for Dart panel. This will generate or update your Dart project.

Download Extension Manager

Download the Toolkit for Dart v0.4.2

3. Code

Code in Dart Editor* with full code completion of your assets and the StageXL Dart library.

Download Dart SDK and tools

* other editors are supported, like IntelliJ IDEA and WebStorm, Sublime Text, VIM,...
